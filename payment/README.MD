# 支付收银台说明文档

 Payment gateway 是StarsPower里的其中一个模块，是管理支付业务的模块。项目集成了Swagger实现在线API文档的功能。

Swagger的在线API文档的功能在开发环境和测试环境可访问，方便开发人员查看与测试，而在正式环境是禁止访问的。
`http://127.0.0.1:8010/swagger-ui.html`
